# VibeLab

> **Build better AI workflows, faster**

---

## Vision

VibeLab is the hub for **AI tool optimization**:
- Security Scanner for GitHub repos
- Universal AI Memory (supermemory-style)
- Prediction Market Intelligence
- Pro blueprints for AI tools
- Universal skills for coding agents

---

## The 5 Pillars

### 1. Security Scanner (`/scan`) 
Enterprise-grade vulnerability scanning.
- 218+ patterns across 6 categories
- GitHub OAuth for private repos
- AI-powered fix suggestions (Gemini)
- Export: JSON, SARIF, Markdown

### 2. AI Memory (`/memory`) 
Supermemory-style universal AI memory.

| Feature | Description |
|---------|-------------|
| Chat | Ask questions, AI answers with your context |
| Semantic Search | AI-powered vector search |
| Memory Graph | Visualize connections between memories |
| URL Import | Fetch and save any webpage |
| File Upload | Import .txt, .md, .json files |
| Browser Extension | Auto-capture from 7 AI sites |
| MCP Server | Access from Cursor/Claude Desktop |

**Token Optimization:**
- Budget slider (500-8000 tokens)
- Auto-summarization (10x compression)
- Tiered storage (Hot/Warm/Cold)

**Browser Extension Sites:**
- ChatGPT
- Claude
- Gemini
- Perplexity
- Microsoft Copilot
- Twitter/X

### 3. Prediction Markets (`/predictions`) 
AI-powered prediction market intelligence.

| Feature | Description |
|---------|-------------|
| Market Scanner | Aggregate from Polymarket & Kalshi |
| Arbitrage Finder | Detect guaranteed profit opportunities |
| AI Research Agent | Deep probability analysis |
| Smart Money Tracker | Track whale wallet activity |
| News Sentiment | AI-powered news sentiment analysis |
| Market Alerts | Custom price and volume alerts |

**How it helps:**
- Find mispriced markets
- Get AI probability estimates
- Detect arbitrage opportunities
- Track high-volume markets
- Follow smart money moves

### 4. Tool Directory 
Pro blueprints for AI tools: Cursor, Midjourney, Kling, Flux, ElevenLabs, etc.

### 5. Universal Skills (`/skills`) 
Coding rules that work across AI agents.
- Export to: Cursor, Claude Code, Windsurf, Antigravity

### 6. VibeLab Loop (`vibeloop/`) 
Autonomous AI development loop ported from [Ralph](https://github.com/frankbria/ralph-claude-code).

| Command | Description |
|---------|-------------|
| `vibeloop` | Run autonomous loop |
| `vibeloop-setup` | Create new project |
| `vibeloop-import` | Convert PRD/specs |
| `vibeloop-monitor` | tmux dashboard |

**Features:**
- Exit Detection (auto-stop when complete)
- Circuit Breaker (prevents stuck loops)
- Session Continuity (24h expiring sessions)
- VS Code Extension with Dashboard

**CLI Adapters:**
- Aider ( Recommended)
- Claude Code ( Ready)
- Cursor, OpenCode ( Experimental)

**Language Model API (VS Code):**
- Antigravity ( via vscode.lm)
- GitHub Copilot ( via vscode.lm)
