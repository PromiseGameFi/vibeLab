# Build & Run Instructions

## Setup
```bash
# Install dependencies
npm install  # or pip install -r requirements.txt
```

## Build
```bash
npm run build  # or python -m build
```

## Run
```bash
npm start  # or python main.py
```

## Test
```bash
npm test  # or pytest
```

## Lint
```bash
npm run lint  # or ruff check .
```
